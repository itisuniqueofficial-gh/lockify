Changelog

Complete version history and release notes for Lockify.

v1.0.0 — Initial Release
2026 · Stable
Added Full app lock protection — lock any installed app with a secure overlay
Added PIN authentication with animated indicators
Added Pattern lock with 3×3 grid and haptic feedback
Added Biometric authentication (fingerprint / face) support
Added Anti-uninstall protection via Device Administrator
Added Forgot password recovery using device lock credentials
Added Unlock time duration — keep apps unlocked for configurable period
Added Trigger exclusions — exclude apps from triggering lock
Added Auto unlock on correct PIN entry (optional)
Added Maximum brightness option on lock screen
Added Haptic feedback toggle
Added Debug logging with export capability
Added Usage Stats backend as alternative detection method
Security Privacy-first design — zero data collection, no cloud sync
Security Secure credential storage using Android SharedPreferences
Security Anti-bypass protection — back press and app switching cannot bypass lock
Performance Zero-lag lock screen overlay with instant trigger
Performance Optimized background service — minimal battery and memory usage
Performance App icon and label caching for smooth list rendering
Performance Removed all Shizuku dependencies for cleaner, lighter build
