# Icons

Place the following icon files here:

- `favicon-16.png` — 16×16 favicon
- `favicon-32.png` — 32×32 favicon
- `apple-touch-icon.png` — 180×180 Apple touch icon
- `icon-192.png` — 192×192 PWA icon
- `icon-512.png` — 512×512 PWA icon

All icons should use the Lockify brand (lock icon on primary blue background).
